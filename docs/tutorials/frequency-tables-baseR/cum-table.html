Cumulative Proportion Table

What might be clearer than the proportion table on the last page are the cumulative frequencies or proportions. This would allow us to see how many songs in total, or what proportion of all songs, were released before a certain year.

To do this, we use the cumsum() function on the tables. First the frequency table:

table.year <- table(songs$YEAR, exclude = NULL)
cumsum(table.year)
